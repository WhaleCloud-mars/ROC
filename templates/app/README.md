# whale app

## 这是使用web可视化工具[Killer-Whale](https://github.com/WhaleCloud-mars/Killer-Whale)编辑，通过[ROC](https://github.com/WhaleCloud-mars/ROC)自动编程，生成的项目。

## 使用
```
$ npm install expo-cli -g 
or
$ yarn global add expo-cli
```
注意：expo-cli尽量保持最新，不然可能会无法运行此项目。
```
$ npm install or yarn
$ npm start or npm run ios
```
